# Crustal Displacement Dynamics Screen

This repository contains an order of mmagnitude physics screen for a proposed 104° coherent crust/lithosphere displacement. The notebook estimates the kinematic, kinetic-energy, angular-impulse, and mean-torque requirements for several moving shell assumptions, including simple spherical shells and LITHO1.0-derived crust/lithosphere mass estimates.

The goal is not to prove or disprove a specific historical event. The goal is to define the physical requirements clearly enough to test whether proposed mechanisms operate at the necessary scale.
